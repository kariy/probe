# `starknet-cast`

A command-line tool for performing RPC calls to the StarkNet network. Similar to the one found in [foundry](https://github.com/foundry-rs/foundry/tree/master/cast).

## Features

-   [x] `--to-dec`
-   [x] `--to-hex`
-   [x] `--from-utf8`
-   [x] `--str-to-felt`
-   [x] `--max-felt`
-   [x] `--max-signed-felt`
-   [x] `--min-signed-felt`
-   [x] `age`
-   [ ] `balance`
-   [x] `block`
-   [x] `block-number`
-   [x] `chain-id`
-   [x] `ecdsa`
-   [ ] `estimate`
-   [ ] `events`
-   [x] `keccak`
-   [x] `nonce`
-   [x] `pedersen`
-   [x] `receipt`
-   [x] `storage`
-   [x] `tx`
-   [x] `tx-count`
-   [x] `tx-pending`
-   [x] `tx-status`
