# `starknet-cast`

Like the one in [foundry](https://github.com/foundry-rs/foundry/tree/master/cast) but for StarkNet.

## Features

-   [x] `--to-dec`
-   [x] `--to-hex`
-   [x] `--from-utf8`
-   [x] `--str-to-felt`
-   [x] `--max-felt`
-   [x] `--max-signed-felt`
-   [x] `--min-signed-felt`
-   [ ] `balance`
-   [ ] `block`
-   [ ] `block-number`
-   [ ] `chain-id`
-   [x] `ecdsa`
-   [ ] `estimate`
-   [ ] `events`
-   [x] `keccak`
-   [x] `pedersen`
-   [ ] `receipt`
-   [ ] `tx`
-   [ ] `tx-status`
